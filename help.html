Z80 Studio Menu Help

||| CONFIG |||

Download Config Template

Downloads a template Z80 Studio configuration file (with comments). Configuration files can be imported with the Import Config option.

Download Config (No Comments)

Downloads a template Z80 Studio configuration file (without comments). Configuration files can be imported with the Import Config option.

Import Config

Allows the upload of a Z80 Studio configuration file. Must be in JSON format. If the uploaded file is invalid or not formatted correctly, nothing will happen and an INVALID CONFIG error will occur.

||| HELP |||

How to Use This Menu

Opens this page.

Z80 Instruction Set

Redirects to clrhome.org's Z80 Instruction Set table. It displays both the hex opcodes as well as ASM names, and provides fairly detailed technical info for each instruction. If you need help with ASM programming, this is the place to go.

||| SAVE/LOAD PROGRAM |||

Save Editor

Saves the contents of the editor as a .txt file.

Save RAM

Saves a copy of RAM as ASCII string (.txt file).

Load Editor

Allows the upload of an editor data file. The file can have any extension. The data read from this file is pasted directly into the editor.
